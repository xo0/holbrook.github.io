Set Slide Background

Slide background can be set to a color, an image or a repeating image array by setting heading properties.

Single Colored Background

Set property revealbackground to either an RGB color value, or any supported CSS color format.

,*** Single Colored Background

Single Image Background

Set property revealbackground to an URL of background image. Set property revealbackgroundtrans to slide to make background image sliding rather than fading.

,*** Single Image Background

Repeating Image Background

Resize background image by setting property revealbackgroundsize to a number.

Set property revealbackgroundrepeat to repeat to repeat image on the background.

,*** Repeating Image Background
