
## 语法高亮

主题文件在~/.vim/colors/中

设置默认主题：

```vim
    
    syntax on " 打开语法高亮
    color monokai " 设置主题

```

## 插件

插件，插件文档和插件语法高亮支持分别放在：

    ~/.vim/plugin/
    ~/.vim/doc/
    ~/.vim/syntax/

管理插件的Vim插件vundle：https://github.com/gmarik/vundle。

vundle可以使得安装的插件的文件都放到同一个目录下，并且简单设置就可使用。安装方法如下：
        首先，在终端输入如下命令：

git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
