处理缺失数据

数据不完整在数据分析的过程中很常见。

pandas使用浮点值NaN表示浮点和非浮点数组里的缺失数据。

pandas使用isnull()和notnull()函数来判断缺失情况。

对于缺失数据一般处理方法为:

  • 滤掉, 使用 dropna()
  • 填充，使用 fillna，比如 ffill(指定值）或 bfill（插值
